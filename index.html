HTML & CSS Cheatsheet

Commonly Used HTML Tags

Tag Name Description
<a> Anchor The element for creating hyperlinks to various places a URL can address.
<span> Span The inline element container used primarily for styling certain content.
<div> Div The element used a generic container for content. Divs need to be styled with CSS.
<head> Head The element used to contain metadata such as stylesheets, page titles, or scripts.
<h1> - <h6> Section Headings The six levels of section headings from largest (h1) to smallest (h6).
<img> Image Embed The element used to embeds images in a document.
<p> Paragraph The element used to represent/contain a paragraph of text.
<footer> Footer The element denoting the footer section of a page.

Commonly Used CSS Properties

Property Name Description
color Color The property that assigns the foreground color of of an element's text and text decorations.
background-color Background Color The property that sets the background color of an element.
font-family Font Family The property that that specifies an ordered list of one or more fonts to display text in.
font-weight Font Weight The property that sets the weight (boldness) of the font.
font-size Font Size The property that sets the size of a font. Also changes the size-relative values of the font.
margin Margin The property used to set the margin area for all four sides of an element.
text-align Text Align The property that sets the horizontal alignement of inline content in a block element or table-cell.
text-shadow Text Shadow The property that adds shadows to text.
